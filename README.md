# Student Performance Prediction

## Project Overview

This project predicts whether a student is an excellent student using Machine Learning.

## Models Used

- Logistic Regression
- Random Forest
- XGBoost

## Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Preprocessing
5. Model Training
6. Cross Validation
7. Model Comparison
8. Feature Importance

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Results

XGBoost achieved the highest ROC-AUC score.
